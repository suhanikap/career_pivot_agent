# ⚡ Career Pivot Agent Engine

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://careerpivotagent-9kemmqjavbwkdbspxsee6x.streamlit.app/)

A full-stack, cloud-integrated application built to map user credentials, qualifications, and backgrounds into tailored modern tech career paths. The application takes non-technical or transitional career histories and processes them through an intelligent AI engine, saving all structured blueprints directly into a cloud database.

## 🚀 Live Deployment & Links

The production build of this application is fully hosted on the cloud and accessible globally:

* **⚡ Live Interactive Portal:** [Career Pivot Portal Interface](https://careerpivotagent-9kemmqjavbwkdbspxsee6x.streamlit.app/)
* **🗄️ Serverless Database Cluster:** Powered by [Neon.tech PostgreSQL](https://neon.tech/)
* **🤖 Core AI Engine:** Backed by Google AI Studio (Gemini)

---

## 🛠️ Tech Stack & Infrastructure

* **Frontend UI:** Streamlit (Python Web Framework)
* **Database Engine:** Serverless Neon.tech PostgreSQL (Relational JSONB Storage)
* **AI Model Pipeline:** Gemini 2.5 Flash API via `google-genai`
* **Security & Configuration:** Encrypted Streamlit Cloud Environment Secrets
